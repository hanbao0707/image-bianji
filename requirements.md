# Image Bianji - MVP 需求文档

## 项目概述

一个纯前端的图片编辑工具，提供 AI 智能抠图、尺寸调整、压缩等功能，无需服务器，所有处理在浏览器端完成。

## 核心功能

### 1. ✂️ 移除背景（AI 抠图）

**描述**：使用 AI 自动识别图片主体并移除背景，输出透明背景图片。

**技术方案**：
- 使用浏览器内置的 `window.createImageBitmap()` + Canvas API
- 或集成第三方纯前端抠图库（如 @imgly/background-removal）
- 处理完全在客户端进行，无需上传服务器

**交互**：
- 点击"移除背景"按钮
- 显示处理进度
- 完成后预览透明背景效果（棋盘格背景显示）

---

### 2. 📐 调整尺寸

**描述**：自定义图片宽度和高度。

**功能**：
- 输入目标宽度/高度
- 支持"保持宽高比"开关
- 实时预览尺寸变化
- 显示当前尺寸信息

**交互**：
- 数字输入框
- 锁定比例按钮
- 预设尺寸快捷选择（如 800x600, 1920x1080）

---

### 3. 🗜️ 压缩图片

**描述**：调节图片压缩质量，减小文件体积。

**功能**：
- 质量滑块：10% - 100%
- 实时显示压缩前后文件大小对比
- 支持选择输出格式（JPEG/PNG/WebP）

**交互**：
- 滑块调节
- 显示：原始大小 → 压缩后大小 → 节省比例
- 预览压缩效果

---

### 4. 🎨 替换背景

**描述**：移除背景后，可快速更换纯色背景。

**功能**：
- 预设颜色：白、黑、红、绿、蓝、黄、灰
- 自定义颜色选择器
- 背景透明度调节

**交互**：
- 颜色按钮快捷选择
- 颜色选择器高级选项
- 实时预览效果

---

### 5. ⬇️ 下载图片

**描述**：一键保存编辑后的图片到本地。

**功能**：
- 支持选择导出格式（PNG/JPEG/WebP）
- 支持选择文件名
- 下载当前编辑结果

---

### 6. 🔄 重置

**描述**：恢复到原始图片，清除所有编辑。

**功能**：
- 一键恢复原图
- 清除所有编辑状态
- 确认提示（防止误操作）

---

## 技术架构

### 纯前端实现

```
┌─────────────────────────────────────┐
│            浏览器                    │
│  ┌─────────────────────────────────┐│
│  │        HTML + CSS               ││
│  │    (响应式 UI 界面)              ││
│  └─────────────────────────────────┘│
│  ┌─────────────────────────────────┐│
│  │        JavaScript               ││
│  │  - Canvas API (图片处理)         ││
│  │  - AI 抠图库 (背景移除)          ││
│  │  - File API (上传下载)           ││
│  └─────────────────────────────────┘│
└─────────────────────────────────────┘
         ↓ 无服务器通信
    用户本地处理
```

### 技术栈

| 模块 | 技术选型 |
|------|---------|
| 框架 | 原生 HTML/CSS/JS 或 Vue 3 |
| 图片处理 | Canvas API |
| AI 抠图 | @imgly/background-removal |
| 构建工具 | Vite（可选） |
| 部署 | 静态托管（GitHub Pages / Vercel） |

### AI 抠图库选型

推荐使用 **@imgly/background-removal**：
- 纯前端运行，无需服务器
- 基于机器学习模型
- 支持 WebAssembly 加速
- 文件大小：约 30MB（首次加载后缓存）

备选方案：
- **rembg-js** - Python rembg 的 JS 移植
- **TensorFlow.js + BodyPix** - 自建模型方案

## UI 设计

### 布局结构

```
┌────────────────────────────────────────────────┐
│                  Image Bianji                   │
│              图片编辑工具 (MVP)                  │
├────────────────────────────────────────────────┤
│  ┌──────────────┐  ┌────────────────────────┐  │
│  │              │  │    工具栏              │  │
│  │              │  │  [移除背景] [调整尺寸] │  │
│  │   预览区域   │  │  [压缩]     [换背景]   │  │
│  │              │  │                        │  │
│  │   (原图/     │  │    参数面板            │  │
│  │    处理后)   │  │  (根据选中工具显示)    │  │
│  │              │  │                        │  │

```

### 响应式设计

- **桌面**：左右分栏，预览区 + 工具栏
- **移动端**：上下布局，预览在上，工具在下
- 拖拽上传区域全局可用

## 用户流程

```
用户进入页面
    ↓
上传图片（拖拽/点击）
    ↓
选择编辑功能
    ├── 移除背景 → AI 处理 → 预览结果
    ├── 调整尺寸 → 输入尺寸 → 预览结果
    ├── 压缩图片 → 调节质量 → 预览结果
    └── 替换背景 → 选择颜色 → 预览结果
    ↓
满意后下载
    ↓
重置或继续编辑
```

## 技术限制

- **AI 抠图**：首次加载模型约 30MB，需等待下载
- **浏览器支持**：现代浏览器（Chrome 90+, Firefox 88+, Safari 14+）
- **文件大小**：建议 < 20MB，大文件处理较慢
- **图片格式**：支持 JPG、PNG、WebP、GIF（静态帧）
