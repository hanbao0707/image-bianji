# Image Bianji - 图片编辑工具 v2.0

## 项目概述
Image Bianji 是一个基于 Next.js + TailwindCSS 的纯前端图片编辑工具，所有图片处理都在浏览器中完成，保护用户隐私。v2.0 版本增加了大量高级功能。

## ✅ 已实现功能

### 📸 基础功能
- **图片上传**: 支持拖拽和点击上传
- **格式支持**: JPG、PNG、WebP，最大 20MB
- **实时预览**: 即时查看编辑效果
- **撤销功能**: 支持多步撤销操作
- **下载功能**: 一键下载编辑后的图片

### 🎨 核心编辑功能
1. **背景移除**: 基础颜色检测算法
2. **调整尺寸**: 支持自定义尺寸和保持宽高比
3. **图片压缩**: 可调节质量 (10-100%)，支持 JPEG、PNG、WebP 输出
4. **格式转换**: JPEG、PNG、WebP 格式互转
5. **图片旋转**: 0°、90°、180°、270° 旋转
6. **图片裁剪**: 预设裁剪和自定义裁剪
7. **背景替换**: 7种预设颜色 + 自定义颜色

### 🎨 高级滤镜功能
- **亮度调节**: 0-200% 可调
- **对比度调节**: 0-200% 可调
- **饱和度调节**: 0-200% 可调
- **模糊效果**: 0-20px 可调
- **黑白效果**: 0-100% 可调
- **复古效果**: 0-100% 可调
- **滤镜预设**: 原图、黑白、复古、鲜艳、暗调、模糊

### ✨ 图片增强功能
- **锐化**: 0-100 可调
- **降噪**: 0-100 可调
- **暗角**: 0-100 可调
- **曝光**: 0-200% 可调
- **增强预设**: 原图、清晰、专业、艺术、暗调

### 🏷️ 水印功能
- **文字水印**: 自定义文字内容
- **字体大小**: 10-100px 可调
- **颜色选择**: 颜色选择器 + 十六进制输入
- **透明度**: 0-100% 可调
- **位置选择**: 左上、右上、左下、右下、居中

### 📦 批量处理功能
- **多文件上传**: 支持 1-10 张图片
- **批量应用滤镜**: 同时应用所有滤镜设置
- **进度显示**: 实时显示处理进度
- **批量下载**: 自动下载所有处理后的图片

### 📊 信息展示
- **图片信息**: 尺寸、文件大小、宽高比、像素总数
- **压缩信息**: 显示压缩前后大小对比
- **进度指示**: 处理进度条和状态提示
- **错误处理**: 友好的错误提示和验证

### 🎯 用户体验
- **响应式设计**: 适配桌面和移动设备
- **直观界面**: 清晰的工具面板和操作按钮
- **实时反馈**: 即时显示处理结果
- **隐私保护**: 所有处理在本地完成
- **快捷操作**: 支持键盘快捷键

## 🔧 技术栈
- **框架**: Next.js 14.2.35 (App Router)
- **样式**: TailwindCSS
- **语言**: TypeScript
- **图片处理**: HTML5 Canvas API
- **构建工具**: Next.js 内置 Webpack

## 🎨 功能详情

### 1. 背景移除
- 使用基础颜色检测算法
- 检测图片四个角的背景色
- 移除相似颜色的像素
- 支持透明背景输出

### 2. 图片调整
- **尺寸调整**: 自定义宽度和高度
- **宽高比**: 保持原始比例选项
- **实时预览**: 调整即时生效

### 3. 图片压缩
- **质量控制**: 10-100% 可调
- **格式支持**: JPEG、PNG、WebP
- **压缩预览**: 显示压缩前后大小对比

### 4. 格式转换
- **JPEG**: 有损压缩，文件小
- **PNG**: 无损压缩，支持透明度
- **WebP**: 现代格式，体积小

### 5. 图片旋转
- **0°**: 无旋转
- **90°**: 顺时针旋转
- **180°**: 上下颠倒
- **270°**: 逆时针旋转

### 6. 图片裁剪
- **预设裁剪**: 左半边、右半边、上半边、下半边
- **居中裁剪**: 80% 居中裁剪
- **自定义裁剪**: 自定义 X、Y、宽度、高度

### 7. 滤镜效果
- **亮度**: 0-200% 可调
- **对比度**: 0-200% 可调
- **饱和度**: 0-200% 可调
- **模糊**: 0-20px 可调
- **黑白**: 0-100% 可调
- **复古**: 0-100% 可调
- **预设**: 原图、黑白、复古、鲜艳、暗调、模糊

### 8. 图片增强
- **锐化**: 提升图片清晰度
- **降噪**: 减少图片噪点
- **暗角**: 添加暗角效果
- **曝光**: 调整图片亮度
- **预设**: 原图、清晰、专业、艺术、暗调

### 9. 水印功能
- **文字水印**: 自定义水印文字
- **字体大小**: 10-100px 可调
- **颜色选择**: 支持颜色选择器和十六进制输入
- **透明度**: 0-100% 可调
- **位置**: 左上、右上、左下、右下、居中

### 10. 批量处理
- **多文件上传**: 支持 1-10 张图片
- **批量应用**: 同时应用所有滤镜和增强设置
- **进度显示**: 实时显示处理进度

## 📊 性能特点

- **浏览器兼容性**: 需要现代浏览器支持
- **批量处理**: 最多 10 张图片

## 📝 使用说明

1. **上传图片**: 拖拽或点击上传区域选择图片
2. **选择工具**: 点击相应工具按钮打开功能面板
3. **调整参数**: 根据需要调整各项参数
4. **预览效果**: 在预览区域查看编辑结果
5. **保存图片**: 点击下载按钮保存编辑后的图片
6. **批量处理**: 使用批量处理功能同时处理多张图片

## 🔒 安全说明

- 所有图片处理都在用户浏览器中完成
- 图片不会上传到任何服务器
- 完全保护用户隐私
- 无需 API 密钥或注册
